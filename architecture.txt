VISION VAULT - PORTFOLIO WEBSITE ARCHITECTURE

COMPONENT ARCHITECTURE
---------------------

   - Smooth scroll navigation to sections
   - Logo/brand name display

2. HERO COMPONENT (src/components/Hero.tsx)

   - Personal introduction and tagline

   - Social media links (GitHub, LinkedIn, Twitter)

3. ABOUT COMPONENT (src/components/About.tsx)
   - Two-column layout: text content + skills preview
   - Personal story and experience description
   - Key statistics grid (projects, experience, clients)
   - Core competencies with progress bars
   - Download resume button

4. PROJECTS COMPONENT (src/components/Projects.tsx)

   - Project cards with hover effects
   - Technology tags for each project
   - Live demo and source code links
   - Project images with overlay actions

5. SKILLS COMPONENT (src/components/Skills.tsx)

   - Animated progress bars with percentages
   - Additional technologies as tags
   - Certifications section with icons
   - Responsive card layout

6. CONTACT COMPONENT (src/components/Contact.tsx)
   - Two-column layout: contact info + form
   - Interactive contact form with validation
   - Contact methods (email, phone, location)
   - Social media links
   - Form submission handling with status messages
   - Responsive design

7. FOOTER COMPONENT (src/components/Footer.tsx)
   - Multi-column footer layout
   - Brand section with description
   - Quick navigation links
   - Services list
   - Social media icons
   - Copyright and legal links
   - "Built with" technology credits

STYLING ARCHITECTURE
-------------------

TAILWIND CSS V4 SETUP:
- Uses @import "tailwindcss" in globals.css
- CSS custom properties for theming
- Dark mode support with prefers-color-scheme

CURRENT COLOR SYSTEM:
- --background: Light (#ffffff) / Dark (#0a0a0a)
- --foreground: Light (#171717) / Dark (#ededed)

CUSTOM COLOR PALETTE INTEGRATION:
Add your portfolio colors to src/app/globals.css in the @theme inline block:

@theme inline {
  /* Existing colors */
  --color-background: var(--background);
  --color-foreground: var(--foreground);
  
  /* ADD YOUR CUSTOM PORTFOLIO COLORS HERE */
  --color-primary: #your-primary-color;        /* Main brand color */
  --color-primary-foreground: #your-text-on-primary;
  --color-secondary: #your-secondary-color;    /* Accent color */
  --color-secondary-foreground: #your-text-on-secondary;
  --color-accent: #your-accent-color;          /* Highlight color */
  --color-accent-foreground: #your-text-on-accent;
  --color-muted: #your-muted-color;            /* Subtle backgrounds */
  --color-muted-foreground: #your-muted-text;
  --color-border: #your-border-color;          /* Borders and dividers */
  --color-input: #your-input-background;       /* Form inputs */
  --color-ring: #your-focus-ring-color;        /* Focus rings */
  
  /* Font families */
  --font-sans: var(--font-geist-sans);
  --font-mono: var(--font-geist-mono);
}

RESPONSIVE DESIGN:
- Mobile-first approach
- Breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px)
- Grid layouts adapt from 1 column to 2-4 columns
- Typography scales appropriately
- Touch-friendly interactive elements

COMPONENT USAGE
---------------

To use these components in your main page (src/app/page.tsx):

import Header from '@/components/Header';
import Hero from '@/components/Hero';
import About from '@/components/About';
import Projects from '@/components/Projects';
import Skills from '@/components/Skills';
import Contact from '@/components/Contact';
import Footer from '@/components/Footer';

export default function Home() {
  return (
    <>
      <Header />
      <main>
        <Hero />
        <About />
        <Projects />
        <Skills />
        <Contact />
      </main>
      <Footer />
    </>
  );
}

CUSTOMIZATION GUIDE
------------------

1. COLORS & BRANDING:
   - Update color variables in globals.css @theme block
   - Replace "Vision Vault" with your brand name in Header and Footer
   - Update logo/brand styling

2. CONTENT:
   - Replace placeholder text with your actual content
   - Update social media links throughout components
   - Add your project data in Projects.tsx
   - Update skills and percentages in Skills.tsx
   - Replace contact information in Contact.tsx

3. IMAGES:
   - Add profile photo to public/ folder
   - Add project screenshots to public/ folder
   - Update image paths in components
   - Optimize images for web (WebP format recommended)

4. FUNCTIONALITY:
   - Implement actual form submission in Contact.tsx
   - Add analytics tracking
   - Implement smooth scrolling behavior
   - Add loading states and animations

5. SEO & METADATA:
   - Update metadata in layout.tsx
   - Add proper alt texts for images
   - Implement structured data
   - Add sitemap and robots.txt

DEVELOPMENT WORKFLOW
-------------------

1. Install dependencies: `pnpm install`
2. Run development server: `pnpm dev`
3. Build for production: `pnpm build`
4. Start production server: `pnpm start`
5. Lint code: `pnpm lint`
